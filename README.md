# mini-automation-service


Basit, deploy edilebilir otomasyon servisi (Flask).


## Özellikler
- `/api/tc-validate` : TC kimlik doğrulama (mock)
- `/api/send-email` : SMTP ile e-posta gönderme
- `/api/health` : sağlık kontrolü
- Planlı görev (APSChduler)


## Hızlı başlatma (lokal)


1. Dosyaları kaydet
2. `.env.example`'ı `.env` olarak kopyala ve içini doldur
